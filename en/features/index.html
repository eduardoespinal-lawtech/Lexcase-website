And everything else

Small things that change your flow.

Global search

Cases, contacts and notes via one shortcut (⌘E). Fuzzy search that tolerates typos.

Keyboard shortcuts

⌘N new · ⌘U contact · ⌘T task · ⌘H hearing · Esc close. Run it all without touching the mouse.

Duplicate detection

Same phone or ID number, even if the name or email differs. 90% similarity match by phone or ID.

Automatic backups

Export your whole database to JSON in the folder you choose — daily, weekly or monthly. Import anytime.

Notes editor

Rich-text editor inspired by Apple Notes on every case. Link local folders or Google Drive folders.

Native design

Windows, traffic lights and system materials. Automatic light and dark mode. Full Spanish and English.

macOS 11 or later · Windows coming soon

Lexbell is available for macOS 11 Big Sur or later. A Windows 11 version is coming soon.
